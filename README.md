# Aery Extensions

Extension packs for [Aery](https://github.com/eminent337/aery) — AI coding agent for the terminal.

## Browse & Install

```bash
aery
/marketplace
```

## Install directly

```bash
aery install eminent337/aery-extensions/packs/full
```
